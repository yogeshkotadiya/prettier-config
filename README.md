# prettier-config

npm package of prettier config

---

## Install

```bash

npm install --dev @yogeshkotadiya/prettier-config

or

yarn add -d @yogeshkotadiya/prettier-config
```

---

## Configuration

```json
{
  "printWidth": 80,
  "tabWidth": 2,
  "trailingComma": "es5",
  "singleQuote": false,
  "semi": true,
  "bracketSpacing": true,
  "jsxBracketSameLine": false,
  "endOfLine": "lf"
}
```
